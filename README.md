# 🔐 Flask User Authentication System

🚀 A modern and beginner-friendly **Flask Web Application** that implements a complete **User Authentication System** with database connectivity and a professional UI.

---

## ✨ Features

✅ User Registration (Create Account)
✅ User Login & Logout (Authentication System)
✅ Session Management 🔑
✅ Update Password 🔄
✅ Delete Account ❌
✅ View User Profile 👤
✅ View All Users (Admin-like Feature) 📋
✅ Flash Messages (Success/Error Alerts) ⚡
✅ Beautiful UI with Gradient Design 🎨

---

## 🛠️ Tech Stack

* 🐍 Python
* 🌐 Flask (Web Framework)
* 🗄️ SQLite (Database)
* 🎨 HTML + CSS (Frontend)

## 🔍 How It Works

* 📝 Users register → Data stored in SQLite database
* 🔐 Login validates user credentials
* 🧾 Session keeps user logged in
* 🔄 Users can update password
* ❌ Users can delete their account
* 📋 Admin-style page shows all users
